# Gotham Rounded Fonts

Bu klasöre aşağıdaki Gotham Rounded font dosyalarını kopyalayın:

- `GothamRnd-Light.otf` (font-weight: 300)
- `GothamRnd-Book.otf` (font-weight: 400)
- `GothamRnd-Medium.otf` (font-weight: 500)
- `GothamRnd-Bold.otf` (font-weight: 700)

Kaynak: myloungers-web klasöründeki GothamRnd*.otf dosyaları

Farklı isimlerde font dosyalarınız varsa globals.css'deki @font-face tanımlarını güncelleyin.
